# 4. CSS Selector

(1) 선택자 종류
    1) 전체 선택자(Universal Selector)
    2) 타입 선택자(Type Selector)
    3) 아이디/클래스 선택자(ID/Class Selector)
    * ID/Class는 알파벳 또는 ‘_’, ‘-’으로 시작해야 함
    4) 선택자 목록(Selector list)      
(2) 우선 순위 계산:         
    * 
    < 타입/가상요소 
    < 클래스/가상클래스/속성 
    < ID 
    < inline-style
